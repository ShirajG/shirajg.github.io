Setting up VIM

This post is going to be all about how to set up Vim for working in Ruby and Javascript. I've been messing around a lot with my Vim workflow lately, and trying to get it to be as fast as possible. I'm going to try to document the really helpful plugins and settings that I'm using to help out anyone in the future who might be looking for this same info.

The first plugin that I've found really useful was ctrlp. This plugin adds Sublime like Ctrl+P functionality for quickly opening files within your project. I've found it to speed up navigation to the same level as Sublime.

The next plugin that I've found really helpful is the amazing YouCompleteMe. This plugin offers great realtime autocompletion in Ruby and Javascript. It can be compiled to have support for the C family of languages as well, but I haven't really bothered with that yet.

Another super handy plugin for Ruby is the endwise plugin which automatically completes do/if/while blocks with and end. This is super quick, and promotes a really fluid workflow when working in Ruby.

In that same vein I have a few mappings in my .vimrc that speed up placing parentheses in Javascript. Here they are:

inoremap {<CR> {<CR>}<Esc>ko

inoremap (<CR> (<CR>)<Esc>ko

inoremap [<CR> [<CR>]<Esc>ko

These mappings automate one of the most tedious tasks in coding, inserting matching parentheses.
